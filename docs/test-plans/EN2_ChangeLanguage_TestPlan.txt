# eNOW Change Language Feature Test Plan

## Requirements Analysis

| Scenario Name | Requirement | Test Case |
|---------------|-------------|-----------|
| Scenario 1: Change Language | 1.1 The Change language screen should display when user clicks on Change language link | 1.1.1 Verify the Change language screen appears when clicking the Change language link |
|  | 1.2 The Change language screen should have correct elements | 1.2.1 Verify Language text box with "Language" label is displayed |
|  |  | 1.2.2 Verify "Save changes" button with configurable color is present and disabled by default |
|  |  | 1.2.3 Verify "Cancel" button with configurable color is present and enabled by default |
|
| Scenario 2: Click on Country Dropdown | 2.1 Language dropdown should display list of all available languages | 2.1.1 Verify clicking on Language dropdown presents list of all available languages |
|  | 2.2 Languages should be displayed in alphabetical order | 2.2.1 Verify languages are displayed in alphabetical order |
|  | 2.3 Selected language should be displayed in the field | 2.3.1 Verify selected language value is displayed in the field |
|  | 2.4 Save changes button should be enabled when language is selected | 2.4.1 Verify "Save changes" button is enabled when user selects a language |
|
| Scenario 3: Click on Cancel Button | 3.1 Cancel button should discard any changes made | 3.1.1 Verify changes are discarded when Cancel button is clicked |
|  | 3.2 Cancel button should dismiss the Change language screen | 3.2.1 Verify Change language screen is dismissed when Cancel button is clicked |
|  | 3.3 Cancel button should reset values to most recently saved values | 3.3.1 Verify values on My account tab are reset to most recently saved values when Cancel is clicked |
|
| Scenario 4: Click on Save Changes Button | 4.1 Save Changes button should save the specified values | 4.1.1 Verify values specified on screen are saved when Save Changes button is clicked |
|  | 4.2 Save Changes button should update all changed values on My Account tab | 4.2.1 Verify all changed values are updated to latest saved values on My Account tab |
|  | 4.3 Save Changes button should dismiss the Change language screen | 4.3.1 Verify Change language screen is dismissed when Save Changes button is clicked |
